Bos Wars

What is Bos Wars?

Bos Wars is a futuristic real-time strategy game. It is possible to play against human opponents over LAN, internet, or against the computer. It successfully runs under Linux and MS Windows. It should run on BSD, MacOS/X, MacOS/Darwin and BeOS.

The newest version of Bos Wars can always be found at http://www.boswars.org.

Installation

Read the installation documentation for installing and building Bos Wars.

Development

Read development.html if you want to help out with Bos Wars development

Documentation on the scripting language (Lua) can be found at http://www.lua.org/manual/5.0/. To configure the game, the engine extends Lua with its scripting API.

The changes done in the Bos Wars project are listed in the file CHANGELOG.

Problems and contact

If you have problems compiling, installing or playing Bos Wars, read faq.html. Please report bugs on our bug tracker.

You can contact us via the website, wiki or on IRC.
