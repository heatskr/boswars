Bos Wars
--------

About
-----

How to install
-------------

Unpack the game data in a games directory:
> tar -xzvf boswars.tgz




How to start the game
---------------------
Execute the folowing command in the stratagus directory:
> ./stratagus -d path-to-games-directory/data.bos
